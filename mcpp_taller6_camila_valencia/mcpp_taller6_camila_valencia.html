Plotting¶

The shape of a limacon can be defined parametrically as:

r = r0 + cos(theta)

x = r cos(theta)

y = r sin (theta)

When r0 = 1, this curve is called a cardioid. Use this definition to plot the shape of a limacon for r0 = 0:8, r0 = 1:0, and r0 = 1:2. Be sure to use enough points that the curve is closed and appears smooth (except for the cusp in the cardioid). Use a legend to identify which curve is which. Save the plot as a .pdf file and send me a copy with your work.

In [17]:
from matplotlib.backends.backend_pdf import PdfPages

theta = np.linspace(0, 2*np.pi, 100)
r = 0.8+np.cos(theta)

x2 = r*np.sin(theta)

plt.plot(x1, x2)
ax.set_aspect(1)
plt.xlabel('$x$')
plt.ylabel('$y$')
plt.title('Limaon r = 0.8');
